# 💥 Взрывной бой

PVP-бот поддерживает продвинутую тактику взрывного боя, используя кристаллы конца и якоря возрождения.

---

## 💎 Кристалл ПВП

Crystal PVP is a high-damage combat technique using End Crystals and Obsidian.

### Как это работает
1. Бот размещает обсидиан рядом с целью
2. Bot places End Crystal on the obsidian
3. Бот взрывает кристалл, нанося огромный урон от взрыва.
4. Бот рассчитывает безопасные расстояния, чтобы избежать самоповреждения.

### Включить Кристальное PVP
```mcfunction
/pvpbot settings crystalpvp true
```

### Требования
Ботам необходимо в своем инвентаре:
- **Обсидиан** - для основы для размещения кристаллов.
- **Кристаллы Края** - для взрывов

### Тактика
- Боты сохраняют безопасную дистанцию ​​от взрывов
- Автоматическое размещение кристаллов и детонация
- Работает во всех измерениях
- Высокий урон (до 20 сердец)

## ⚓ Якорное ПВП

Anchor PVP использует якоря возрождения в качестве взрывного оружия в Overworld и End.

### Как это работает
1. Бот размещает якорь возрождения рядом с целью.
2. Бот заряжает якорь светящимся камнем.
3. Бот пытается установить спавн (вызывает взрыв)
4. Огромный урон близлежащим объектам.

### Включить привязку PVP
```mcfunction
/pvpbot settings anchorpvp true
```

### Требования
Ботам необходимо в своем инвентаре:
- **Якорь возрождения** - взрывное устройство.
- **Светящийся камень** — для зарядки якоря.

### Важные примечания
- Работает только в Overworld и End (не в Nether)
- В Пустоте якоря работают нормально (без взрыва)
- Очень высокий урон
- Расходует якорь и светящийся камень за одно использование.

| `anchorpvp` | bool | false | Enable Anchor PVP |

---

## 💡 Usage Tips

### Crystal PVP
- Give bots stacks of obsidian and crystals
- Лучше всего работает на средней дистанции (5-10 блоков)
- Очень эффективен против бронированных противников.
- Может пробивать щиты

### Anchor PVP
- Дороже, чем Crystal PVP (потребляет якорь при каждом использовании)
- Чрезвычайно высокий урон
- Лучше всего использовать в качестве завершающего приема.
- Запаситесь светящимся камнем

### Комбинирование тактик
```mcfunction
# Enable all explosive combat
/pvpbot settings crystalpvp true
/pvpbot settings anchorpvp true

# Give bot supplies
/give @e[type=player,name=Bot1] obsidian 64
/give @e[type=player,name=Bot1] end_crystal 64
/give @e[type=player,name=Bot1] respawn_anchor 16
/give @e[type=player,name=Bot1] glowstone 64
```

## 🛡️ Безопасность

Боты автоматически:
- Рассчитать безопасное расстояние взрыва
- По возможности избегайте самоповреждения.
- Отдавайте приоритет нанесению урона цели, а не самосохранению.
- Используйте тотемы бессмертия, если они доступны.
