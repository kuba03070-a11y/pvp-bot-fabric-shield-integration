# ⚔️ Боевая система

PVP Bot оснащен продвинутым боевым искусственным интеллектом, который может использовать различное оружие и тактику.

---

## 🗡️ Типы оружия

### Ближний бой
- **Мечи** — Быстрые атаки, хороший урон.
- **Топоры** - Медленнее, но могут сломать щиты.
- Боты автоматически переключаются на рукопашный бой, когда враги близко

### Дальний бой
- **Луки** — натягивайте и отпускайте стрелы.
- **Арбалеты** - Зарядные и огненные болты
- Боты держат оптимальную дистанцию ​​(8-20 блоков)

### Бой с булавой
- **Булава + Ветер** — Атаки в прыжке, наносящие огромный урон.
- Боты используют ветровые заряды для запуска в воздух.
- Разрушительные падающие атаки

### Кристалл ПВП
- **Кристаллы Края + Обсидиан** - Поместите обсидиан и взорвите кристаллы.
- Боты рассчитывают безопасную дистанцию ​​взрыва
- Автоматическое размещение кристаллов и детонация
- Взрывной бой с высоким уроном

### Якорь ПВП
- **Якорь возрождения + Светящийся камень** - Взрывное оружие в Overworld/End
- Боты заряжают якоря светящимся камнем.
- Взорвать, чтобы нанести огромный урон
- Работает только за пределами Пустоты

## 🎯 Таргетинг

### Режим мести
Когда бот получает урон, он автоматически нацеливается на атакующего.
```mcfunction
/pvpbot settings revenge true
```

### Автонацеливание
Боты автоматически ищут врагов в пределах видимости.
```mcfunction
/pvpbot settings autotarget true
```

### Ручная цель
Заставьте бота атаковать определенную цель.
```mcfunction
/pvpbot attack BotName TargetName
```

### Целевые фильтры
Выберите, на каких ботов могут быть нацелены:
```mcfunction
/pvpbot settings targetplayers true   # Target players
/pvpbot settings targetmobs true      # Target hostile mobs
/pvpbot settings targetbots true      # Target other bots
```

## 🛡️ Защита

### Auto-Shield
Боты автоматически поднимают щиты при атаке врагов.
```mcfunction
/pvpbot settings autoshield true
```

### Разрушение щита
Боты используют топоры для отключения щитов противника.
```mcfunction
/pvpbot settings shieldbreak true
```

### Авто-Тотем
Боты держат тотемы бессмертия в руке.
```mcfunction
/pvpbot settings autototem true
/pvpbot settings totempriority true  # Prioritize totem over shield
```

### Auto-Mend
Bots automatically repair damaged armor using XP bottles.
```mcfunction
/pvpbot settings automend true
/pvpbot settings menddurability 0.5  # Repair at 50% durability
```

## 🍎 Healing

### Автопитание
Боты едят еду, когда:
- Здоровье низкое (< 30%)
- Голод ниже порога

```mcfunction
/pvpbot settings autoeat true
/pvpbot settings minhunger 14
```

### Авто-зелья
Боты автоматически используют зелья:
- **Зелья лечения** - при низком уровне здоровья (брызги или питьевые)
- **Зелья силы** - при вступлении в бой.
- **Speed potions** - when entering combat
- **Зелья сопротивления огню** - при вступлении в бой.

Все усиливающие зелья бросаются сразу в начале боя. Боты повторно накладывают усиления по истечении срока действия эффектов (осталось менее 5 секунд).

```mcfunction
/pvpbot settings autopotion true
```

### Отступление
Когда здоровье низкое, боты отступают во время еды/лечения.
Отступление отключено, если у бота нет еды (бьется насмерть).

```mcfunction
/pvpbot settings retreat true
/pvpbot settings retreathp 0.3  # 30% HP
```

## 💥 Критические удары

Боты могут наносить критические удары, рассчитывая свои атаки на прыжки.
```mcfunction
/pvpbot settings criticals true
```

---

## 🕸️ Тактика паутины

Боты могут использовать паутину стратегически:
- **При отступлении** — подкладывает паутину под преследующего врага, чтобы замедлить его.
- **В рукопашном бою** — подкладывает паутину под нападающего врага.

```mcfunction
/pvpbot settings cobweb true
```
