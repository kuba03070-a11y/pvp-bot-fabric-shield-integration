# 🚀 Трюк с надкрыльями

Трюк с надкрыльями — это усовершенствованная техника воздушного боя, сочетающая полет надкрылий с атаками булавами, наносящими разрушительный урон при падении.

---

## 🎯 Как это работает

The ElytraMace trick follows these steps:

1. **Проверка оборудования** – бот проверяет, есть ли у него надкрылья, булава и фейерверки.
2. **Взлет** – бот экипирует надкрылья и использует фейерверк, чтобы набрать высоту.
3. **Positioning** - Bot flies to optimal position above target
4. **Атака** — бот удаляет надкрылья в воздухе и атакует булавой, нанося огромный урон от падения.

---

## ⚙️ Настройки

| Настройка | Тип | Диапазон | По умолчанию | Описание |
|---------|------|-------|---------|-------------|
| `elytramace` | bool | - | true | Enable ElytraMace trick |
| `elytramaceretries`| интервал | 1-10 | 1 | Максимальное количество попыток повторного взлета |
| `elytramacealtitude`| интервал | 5-50 | 20 | Минимальная высота для атаки |
| `elytramacedistance`| двойной | 3-15 | 8.0 | Расстояние атаки от цели |
| `elytramacefireworks`| интервал | 1-10 | 3 | Количество фейерверков для использования |

---

## 🔧 Примеры конфигурации

### Включить ElytraMace
```mcfunction
/pvpbot settings elytramace true
```

### Увеличивайте высоту, чтобы наносить больше урона
```mcfunction
/pvpbot settings elytramacealtitude 30
```

### Разрешить больше попыток взлета
```mcfunction
/pvpbot settings elytramaceretries 3
```

### Используйте больше фейерверков для более высокого полета
```mcfunction
/pvpbot settings elytramacefireworks 5
```

## 📋 Требования

Для работы ElytraMace ботам необходимо:

- **Надкрылья** - В слоте для брони (сундук)
- **Булава** - В инвентаре (любой слот)
- **Фейерверк** - В инвентаре (любой слот)
- **Цель** - В пределах дальности обнаружения

---

## 🎯 Боевой приоритет

ElytraMace имеет **самый высокий приоритет** при выборе оружия:

1. **Надкрылья** (при наличии надкрылий + булавы + фейерверков)
2. Кристалл ПВП (при наличии кристаллов+обсидиан)
3. Anchor PVP (if anchor + glowstone available)
4. Mace + Wind Charge (if mace + wind charges available)
5. Ranged weapons (bow/crossbow)
6. Оружие ближнего боя (меч/топор)

## 🚨 Troubleshooting

### Bot doesn't take off
- Check if bot has elytra equipped in chest slot
- Verify bot has fireworks in inventory
- Ensure `elytramace`настройка включена
- Проверьте, достаточно ли у бота места для прыжка

###Бот падает, не атакуя
- Increase `elytramacealtitude`параметр
- Проверьте, есть ли у бота в инвентаре булава
- Убедитесь, что цель находится в пределах`elytramacedistance`

### Бот продолжает повторять попытки взлета
- Увеличивать`elytramaceretries`параметр
- Проверьте наличие препятствий, блокирующих взлет.
- Убедитесь, что у бота достаточно фейерверков.

---

## 💡 Советы

- **Больше высота = больше урона** - Увеличьте высоту для разрушительных атак.
- **Несколько фейерверков**. Используйте больше фейерверков, чтобы летать выше и быстрее.
- **Свободное пространство**. Убедитесь, что у ботов есть открытое небо для взлета.
- **Управление запасами**. Держите надкрылья, булаву и фейерверки на складе.
- **Позиционирование цели**. Лучше всего работает против неподвижных или медленных целей.

---

## ⚠️ Ограничения

- Для взлета требуется открытое небо
- Потребляет фейерверк при каждом использовании.
- Может выйти из строя в закрытых помещениях.
- Надкрылья получают урон прочности
- Менее эффективен против быстро движущихся целей.
