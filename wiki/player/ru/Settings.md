# ⚙️ Настройки

Полный список всех опций конфигурации.

---

## 📋 Команды

```mcfunction
# Show all settings
/pvpbot settings

# Show specific setting
/pvpbot settings <name>

# Change setting
/pvpbot settings <name> <value>
```

---

## ⚔️ Настройки боя

| Настройка | Тип | Диапазон | По умолчанию | Описание |
|---------|------|-------|---------|-------------|
| `combat`| бул | - | правда | Включить/отключить боевую систему |
| `revenge`| бул | - | правда | Атакуйте сущности, наносящие урон боту |
| `autotarget`| бул | - | ложный | Автоматический поиск врагов |
| `targetplayers`| бул | - | правда | Может ориентироваться на игроков |
| `targetmobs`| бул | - | ложный | Может атаковать враждебных мобов |
| `targetbots`| бул | - | ложный | Может атаковать других ботов |
| `criticals`| бул | - | правда | Выполнить критические удары |
| `ranged`| бул | - | правда | Используйте луки/арбалеты |
| `mace`| бул | - | правда | Используйте булаву с ветровыми зарядами |
| `spear`| бул | - | ложный | Использовать копье (отключено из-за ошибки с ковром) |
| `crystalpvp`| бул | - | ложный | Использовать кристалл ПВП (обсидиан + кристаллы) |
| `anchorpvp`| бул | - | ложный | Использовать якорь PVP (якорь возрождения + светящийся камень) |
| `elytramace`| бул | - | правда | Используйте трюк с надкрыльями (надкрылья + булава) |
| `attackcooldown`| интервал | 1-40 | 10 | Тики между атаками |
| `meleerange`| двойной | 2-6 | 3,5 | Расстояние атаки в ближнем бою |
| `movespeed`| двойной | 0,1-2,0 | 1.0 | Множитель скорости движения |
| `viewdistance`| двойной | 5-128 | 64 | Максимальная дальность обнаружения цели |
| `retreat`| бул | - | правда | Отступить при низком уровне здоровья |
| `retreathp`| двойной | 0,1-0,9 | 0,3 | Процент HP для начала отступления (30%) |

## 🧪 Настройки зелий

| Настройка | Тип | Диапазон | По умолчанию | Описание |
|---------|------|-------|---------|-------------|
| `autopotion`| бул | - | правда | Автоматическое использование лечебных/баффирующих зелий |
| `cobweb`| бул | - | правда | Используйте паутину, чтобы замедлить врагов |

Боты автоматически используют:
- **Зелья лечения** при низком уровне здоровья.
- **Зелья силы** при входе в бой.
- **Зелья скорости** при входе в бой.
- **Зелья сопротивления огню** при входе в бой.
- **Паутина** для замедления врагов (при отступлении или атаке врага)

Все усиливающие зелья бросаются сразу в начале боя или по истечении срока действия эффектов.

## 🚀 Настройки ElytraMace

| Настройка | Тип | Диапазон | По умолчанию | Описание |
|---------|------|-------|---------|-------------|
| `elytramace`| бул | - | правда | Включить трюк ElytraMace |
| `elytramaceretries`| интервал | 1-10 | 1 | Максимальное количество попыток повторного взлета |
| `elytramacealtitude`| интервал | 5-50 | 20 | Минимальная высота для атаки |
| `elytramacedistance`| двойной | 3-15 | 8.0 | Расстояние атаки от цели |
| `elytramacefireworks`| интервал | 1-10 | 3 | Количество фейерверков для использования |

**Хитрость с надкрыльями:** бот экипирует надкрылья, использует фейерверки для взлета, удаляет надкрылья в воздухе и атакует булавой, нанося огромный урон при падении.

## 💾 Файлы конфигурации

Настройки сохраняются в:
```
config/pvp_bot.json
```

Данные бота (позиции, размеры, режимы игры) сохраняются в:
```
config/pvp_bot_bots.json
```

И настройки, и боты сохраняются после перезапуска сервера. Боты автоматически восстанавливаются при запуске сервера.

## 📋 Примеры

### Сделайте ботов более реалистичными
```mcfunction
/pvpbot settings misschance 15
/pvpbot settings mistakechance 10
/pvpbot settings reactiondelay 5
```

### Сделайте ботов агрессивными
```mcfunction
/pvpbot settings autotarget true
/pvpbot settings targetplayers true
/pvpbot settings targetbots true
/pvpbot settings revenge true
```

### Отключить дальний бой
```mcfunction
/pvpbot settings ranged false
/pvpbot settings mace false
/pvpbot settings crystalpvp false
/pvpbot settings anchorpvp false
```

### Быстрое движение
```mcfunction
/pvpbot settings bhop true
/pvpbot settings bhopcooldown 8
/pvpbot settings jumpboost 0.2
/pvpbot settings movespeed 1.5
```

### Стационарные охранники
```mcfunction
/pvpbot settings idle false
/pvpbot settings bhop false
```

### Включить трюк ElytraMace
```mcfunction
/pvpbot settings elytramace true
/pvpbot settings elytramacealtitude 25
/pvpbot settings elytramaceretries 2
```

### Включить команды движения с помощью баритона
```mcfunction
/pvpbot settings gotousebaritone true
```

### Включить специальные имена
```mcfunction
/pvpbot settings specialnames true
```
