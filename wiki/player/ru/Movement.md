# 🚶 Movement System

PVP Bot имеет усовершенствованную систему движения, которая позволяет ботам следовать за игроками, сопровождать цели и перемещаться по определенным координатам.

---

## 🎯 Movement Modes

### Follow Mode
Bots follow a target while maintaining optimal distance.

```mcfunction
# Make bot follow a player
/pvpbot follow Bot1 Steve

# Make bot follow another bot
/pvpbot follow Bot1 Bot2
```

**Behavior:**
- Maintains 3-block distance from target
- Автоматически обходит препятствия
- Следит за измерениями
- Использует интеллектуальный поиск пути
- Останавливается, когда цель находится в автономном режиме

### Режим сопровождения
Улучшенный режим следования с защитными возможностями.

```mcfunction
# Make bot escort a player
/pvpbot escort Bot1 Steve

# Make bot escort another bot
/pvpbot escort Bot1 Bot2
```

**Поведение:**
- То же, что и режим следования
- **Автоматически защищает цель** при нападении
- Приоритет защиты цели над другими боевыми действиями.
- Атакует любого, кто наносит урон сопровождаемой цели.

### Перейти в режим
Прямое перемещение по координатам.

```mcfunction
# Send bot to coordinates
/pvpbot goto Bot1 100 64 200

# Use relative coordinates
/pvpbot goto Bot1 ~10 ~ ~-5
```

**Поведение:**
- Умный поиск пути к координатам
- Обход препятствий и прыжки
- Дополнительная интеграция баритона
- Останавливается, когда пункт назначения достигнут

---

## 🛑 Остановка движения

```mcfunction
# Stop any movement mode
/pvpbot stopmovement Bot1
```

Эта команда останавливает:
- Режим следования
- Режим сопровождения
- Перейти в режим
- Следование по пути

---

## 👥 Движение фракций

Все команды движения работают с фракциями:

```mcfunction
# Make entire faction follow a target
/pvpbot faction follow RedTeam Steve

# Make entire faction escort a target
/pvpbot faction escort RedTeam Steve

# Move entire faction to coordinates
/pvpbot faction goto RedTeam 100 64 200
```

## ⚙️ Настройки движения

| Настройка | Тип | Диапазон | По умолчанию | Описание |
|---------|------|-------|---------|-------------|
| `movespeed`| двойной | 0,1-2,0 | 1.0 | Базовая скорость передвижения |
| `bhop`| бул | - | правда | Включить банни-хоп |
| `bhopcooldown`| интервал | 5-30 | 12 | Тики между прыжками |
| `jumpboost`| двойной | 0,0-0,5 | 0,0 | Дополнительная высота прыжка |
| `gotousebaritone`| бул | - | ложный | Используйте баритон для перехода |

## 🧭 Интеграция баритона

Включите Baritone для расширенного поиска пути:

```mcfunction
/pvpbot settings gotousebaritone true
```

**Преимущества:**
- Сложная навигация по местности
- Поиск пути на большом расстоянии
- Автоматическое построение моста
- Добыча через препятствия

**Требования:**
- Должен быть установлен мод Baritone
- Работает только с командами перехода
- Может быть медленнее, чем базовый поиск пути.

## 🔧 Технические подробности

### Следовать на расстоянии
- **Целевое расстояние:** 3 блока.
- **Тормозное расстояние:** 2 квартала.
- **Максимальное расстояние:** 50 блоков (при превышении телепортируйтесь)

### Частота обновления
- Движение обновляется каждый тик (20 раз в секунду)
- Целевая позиция проверяется каждые 5 тиков
- Поиск пути пересчитывается при необходимости.

### Обработка препятствий
- Автоматический прыжок через препятствия длиной в 1 блок.
- Поиск пути вокруг стен и барьеров
- Плавание по воде
- Подъем по лестницам и виноградным лозам

## 🚨 Устранение неполадок

### Бот не подписан
- Проверьте, существует ли цель и находится ли она в сети.
- Убедитесь, что бот не завис и не заблокирован.
- Убедитесь, что движение не отключено
- Проверьте, находится ли бот в бою (может игнорировать движение)

### Бот застревает
- Использовать`/pvpbot stopmovement`и перезапустить
- Включить баритон для сложной местности
- Проверьте наличие препятствий, блокирующих путь.
- Телепортировать бота, чтобы очистить территорию

###Эскорт не защищается
- Убедитесь, что режим сопровождения активен (а не просто следите)
- Проверьте, отключен ли дружественный огонь
- Убедитесь, что у бота включено оружие и бой.
- Цель должна фактически получить урон, чтобы активировать защиту.

## 💡 Советы по использованию

### Эффективное отслеживание
- Для достижения наилучших результатов используйте на открытых площадках.
- Избегайте многолюдной или сложной местности.
- Держите цели движущимися с разумной скоростью.
- Используйте сопровождение для важных игроков

### Координатная навигация
- Используйте goto для точного позиционирования.
- Включить баритон для больших расстояний
- Проверьте, доступны ли координаты
- Используйте относительные координаты для ближайшего движения

### Координация фракций
- Организуйте большие группы с командами фракций.
- Используйте сопровождение для VIP-защиты.
- Координировать атаки с фракцией goto
- Распределите прибытие с помощью шахматных команд
