# Исправление проблемы с бесконечным движением ботов

## Проблема
Боты бесконечно шли вперед при использовании команд `follow`, `escort` и `goto`, не останавливаясь при достижении цели.

## Причина
1. **Отсутствие остановки при достижении цели** - В `BotMovement.updateFollow()` не было команды остановки когда бот достигал нужной дистанции от цели
2. **Неполная остановка в HerobotMovement** - Метод `stopMovement()` не отключал спринт
3. **Постоянное движение вперед** - Бот получал команду `player move forward` каждый тик, но никогда не получал команду остановиться

## Исправления

### 1. BotMovement.java
- **updateFollow()**: Добавлена остановка движения когда бот достаточно близко к цели (< 3 блоков)
- **updateGoto()**: Упрощена логика, добавлена остановка при достижении цели
- **stop()**: Улучшен метод остановки с вызовом HerobotMovement.stopMovement()
- **Частота обновлений**: Увеличена с 10 до 20 раз в секунду (каждые 50мс вместо 100мс)

### 2. HerobotMovement.java
- **walkTowards()**: Добавлена команда `unsprint` когда бот близко к цели (< 3 блоков)
- **stopMovement()**: Добавлена команда `unsprint` для полной остановки движения

## Логика работы

### Follow/Escort режим:
1. Если расстояние до цели > 3 блоков → двигаться к цели
2. Если расстояние до цели ≤ 3 блоков → остановиться
3. При потере цели → остановиться

### Goto режим:
1. Если расстояние до координат > 2 блоков → двигаться к цели
2. Если расстояние до координат ≤ 2 блоков → остановиться и завершить задачу

### Команды HeroBot:
- `player <name> move forward` - движение вперед
- `player <name> sprint` - включить спринт (если далеко)
- `player <name> unsprint` - выключить спринт (если близко)
- `player <name> stop` - остановить все движения
- `player <name> look <yaw> <pitch>` - повернуться к цели

## Тестирование
После исправлений боты должны:
1. ✅ Следовать за целью и останавливаться на расстоянии 3 блоков
2. ✅ Идти к координатам и останавливаться при достижении
3. ✅ Полностью останавливаться при команде `/pvpbot stopmovement`
4. ✅ Не двигаться бесконечно вперед

## Команды для тестирования
```mcfunction
# Создать бота
/pvpbot spawn TestBot

# Тест follow
/pvpbot follow TestBot <ваше_имя>

# Тест goto
/pvpbot goto TestBot 100 64 200

# Остановить движение
/pvpbot stopmovement TestBot
```
